## Changes

M3(시각화 심화) 1차 사이클 — wireframe 블록이 M2 fallback에서 실제 렌더러로
전환됐다. previs 정체성인 `--wf-*` 손그림 체계의 단일 정의 지점이 코드로
확정됐다.

- **`--wf-*` 콘텐츠 토큰 층**: `wireframe.css`에 종이/잉크 톤 토큰(라이트 +
  `.dark` 반전)과 `.wf-root` 시맨틱 요소 스타일을 정의. DESIGN.md 앱 크롬
  토큰과 값을 공유하지 않는다 (시각 계층 분리).
- **Sanitize**: DOMPurify 태그·속성 이중 allowlist (`ALLOW_DATA_ATTR:
  false`). 폼 컨트롤 disabled 강제, `<a>` href·`<img>` src 제거(placeholder
  치환), `.wf-root` 래퍼에서 submit·앵커 클릭 위임 차단 — view-only 보장.
- **Surface 프리셋 5종**: browser/desktop/mobile/popover/panel 프레임 크롬.
  논리 폭·크롬 구성은 렌더러 소유.
- **rough.js 스케치 오버레이**: block.id FNV-1a 해시 시드로 결정적 스케치,
  스트로크·채움은 CSS 변수 참조로 테마 전환 시 재그리기 없이 자동 반전.
  ResizeObserver feature-detect (jsdom 안전).
- **Lazy 로딩 + 에러 바운더리**: wireframe 렌더러를 React.lazy 청크로 분리,
  청크 로드 실패는 블록 단위 에러 바운더리로 격리하고 새로고침 복구 제공
  (동일 specifier 재-import는 브라우저 모듈 맵 캐시로 복구 불가).
- **픽스처**: kitchen-sink에 surface 5종 시연 블록, sample-plan에 계획 검토
  화면 와이어프레임 추가.
- **Tests**: 30건 추가 (총 106건) — sanitize 정책, surface 크롬, 오버레이
  순수 로직, 청크 실패 회귀. 라이트/다크 Playwright 수동 검증 완료.
